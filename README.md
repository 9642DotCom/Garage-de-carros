# Site para venda de carros. O dono da garagem possui uma pagina de admin, pode pode cadastrar os veiculos e alterar o status para disponivel/indisponivel.

O cliente recebe um link onde pode consultar os carros disponiveis, ver fotos e descricao de cada veiculo. Filtrar por preco e etc. 

## Project info
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
